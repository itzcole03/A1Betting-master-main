```bash

```

### **Option 2: Manual Startup**

**Start Backend:**

```bash
cd backend
python main.py
```

- Backend runs on: <http://localhost:8000>
- API docs available at: <http://localhost:8000/docs>

**Start Frontend:**

```bash
cd frontend
npm install
npm run dev
```

- Frontend runs on: <http://localhost:5173>
